```

Example:
"I help mid-sized financial and enterprise software firms modernize legacy systems 
using AI-augmented delivery. Unlike teams that add AI as a layer on top, I integrate 
ML inference, RAG pipelines, and agentic tooling directly into the migration architecture. 
My current focus is brokerage and OMS systems — where domain knowledge in order flow 
combines with AI delivery to cut modernization timelines by 30–40%."
```
